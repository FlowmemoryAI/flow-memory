# Learning Limitations

Flow Memory public alpha learning is local and prototype-grade.

- Memory learning is trace accumulation and retrieval, not guaranteed factual improvement.
- RL learning occurs in simulated Flow Arena environments, not live production systems.
- Neural models are optional and advisory only.
- Tiny training scripts are smoke tests, not production model training.
- V-JEPA 2 and VideoMAE remain adapter seams until real checkpoints are installed and evaluated.
- No real funds, private keys, or live chain actions are used for learning demos.
- Safety policy and approval gates remain authoritative over learned scores or recommendations.
